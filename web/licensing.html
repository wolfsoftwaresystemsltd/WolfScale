üìã What Is the Functional Source License?

The Functional Source License (FSL) is a modern license designed to balance two goals:

  1. Openness ‚Äî the source code is published and freely available
  2. Sustainability ‚Äî the creators can sustain development without their work being resold by competitors

It was created by Sentry and is used by a growing number of companies who want to share their code without giving competitors a free ride.

  • Modify the source code to suit your needs

ü§î What Counts as "Competing"?

The FSL defines a Competing Use as making the software available to third parties as a commercial product or service that is substantially similar to, or a substitute for, any Wolf Software product.

‚ùì Frequently Asked Questions

Is this open source?

The source code is publicly available and you can read, modify, and distribute it. However, it has a permanent competing-use restriction, so it is source-available rather than open source by the OSI definition.

Is it free?

Yes. There is no cost to use, download, modify, or deploy any part of the Wolf software suite. The only restriction is that you cannot use it to offer a competing commercial product.

Can I use it for my company?

Absolutely. If you're using WolfStack to manage your company's servers, WolfNet to connect your offices, or WolfScale to replicate your databases, that's exactly what it's for.

Can I modify the code?

Yes. Fork it, patch it, extend it. You can modify the source code for any non-competing use. If you'd like to contribute your changes back, pull requests are welcome.

Can I redistribute it?

Yes, with attribution. You must retain the copyright notice and licence information. You cannot strip out the Wolf Software branding and resell it as your own product.

Will the licence ever change?

The FSL-1.1 licence applies permanently to all releases. The competing-use restriction does not expire. Wolf Software reserves the right to release future versions under different terms, but existing releases will always remain under FSL-1.1.
