Configuration

Complete configuration reference

Example Config

wolfscale.toml
[node]
id = "node-1"
bind_address = "0.0.0.0:7654"
advertise_address = "10.0.10.10:7654"

[database]
host = "localhost"
port = 3306
user = "wolfscale"
password = "your-password"

[cluster]
auto_discovery = true
# cluster_name = "my-cluster"  # Optional: isolate clusters on same network
# peers = ["10.0.10.11:7654"]  # Optional: seed nodes for faster discovery

[node] Section

Key Required Description
id Yes Unique node ID
bind_address Yes Cluster listen address
advertise_address Yes External address
data_dir No WAL storage directory

[database] Section

Key Required Description
host Yes MariaDB host
port Yes MariaDB port
user Yes DB user
password Yes DB password
pool_size No Pool size (default 10)

[cluster] Section

Key Default Description
peers [] Seed node addresses (optional with auto-discovery)
auto_discovery true Enable UDP broadcast auto-discovery
cluster_name (empty) Optional name to isolate clusters on same network
heartbeat_interval_ms 500 Heartbeat frequency
