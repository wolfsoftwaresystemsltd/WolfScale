Overview

The Issues Scanner runs a comprehensive set of health checks on the selected server and reports any problems found. Click Scan to run all checks ‚Äî results appear in seconds, grouped by severity.

For multi-server clusters, the scanner can also reach remote nodes. If a remote node is unreachable, it's gracefully reported as offline rather than crashing the scan.

ü§ñ AI Analysis

After the scan completes, you can click Analyse with AI to send the issue report to the configured AI provider (Claude or Gemini). The AI will:

  • Explain each issue in plain language
  • Suggest specific remediation steps
  • Prioritise which issues to fix first
  • Flag any issues that might be related to each other
üí° Requires AI Key: AI analysis needs an API key configured in the AI Agent settings. Without it the scan still works ‚Äî you just won't see the AI summary.

üåê Remote Node Scanning

When scanning a remote node, WolfStack proxies the scan request. If the remote node returns a 404 (e.g. running an older version without the issues endpoint), the scanner gracefully reports that the node doesn't support scanning instead of showing an error.
