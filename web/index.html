WolfStack

from Wolf Software Systems Ltd

WolfStack is an all-in-one server and workstation management platform.

  • Create and manage WolfStack clusters on bare metal or existing servers
  • Install on your existing Proxmox VE servers.
  • View multiple clusters in one place, WolfStack clusters and Proxmox Clusters
  • Control everything from a single, beautiful dashboard.
  • Monitor metrics, manage Docker and LXC containers, VMs, R2 Cloud Storage, disk sharing, edit configs across your entire fleet.
  • Perfect for servers, Linux workstations, Rasperry PI, IBM Power.
  • Auto adapts to whatever brand of linux you are running.
  • Will manage one machine to hundreds.
  • Built in secure WolfNet provides a military grade private network.

  1. Install WolfStack on each server ‚Äî run this on every machine you want to manage:
    bash 
    curl -sSL https://raw.githubusercontent.com/wolfsoftwaresystemsltd/WolfStack/master/setup.sh | sudo bash
  2. Get the token from each server ‚Äî after installation, each server displays its cluster token. You can also run wolfstack --show-token
  3. Open the web UI on one server ‚Äî navigate to http://your-server-ip:8553 and log in. You only need to log in to one server.
  4. Add your other nodes ‚Äî click the + button to add each server or Proxmox server. You're done!
  5. Once you have added all the nodes, update WolfNet connections ‚Äî go into your cluster settings and click üîó Update WolfNet Connections to automatically set up peer-to-peer networking between all your nodes.

Note: You can add multiple WolfStack or Proxmox clusters to WolfStack's dashboard for a one-stop server management shop.

WolfDisk
Available Now

Distributed filesystem that shares and replicates files across your network. Mount a shared directory on any number of Linux machines and have your data automatically synchronised. Supports leader, follower and client modes for flexible deployments.
