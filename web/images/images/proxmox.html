Proxmox VE Integration

Seamlessly manage your Proxmox clusters alongside WolfStack nodes from a single pane of glass.

Overview

WolfStack provides deep integration with Proxmox VE, allowing you to monitor nodes, manage VMs and LXC containers, and handle storage and backups directly from the WolfStack dashboard.

‚ú® Key Features

  • ‚úÖ Unified Dashboard: View Proxmox nodes and WolfStack servers side-by-side.
  • ‚úÖ Cluster Support: Group Proxmox nodes by cluster name automatically.
  • ‚úÖ VM & CT Management: Start, stop, reboot, and access consoles for QEMU VMs and LXC containers.
  • ‚úÖ No Agent Required: Connects securely via the Proxmox API using an API Token.

Step 1: Create an API Token in Proxmox

To connect WolfStack to your Proxmox cluster, you need to create an API Token with sufficient privileges.

  1. Log in to your Proxmox VE web interface.
  2. Go to Datacenter > Permissions > API Tokens.
  3. Click Add.
  4. Select a user (e.g., root@pam) and enter a Token ID (e.g., wolfstack).
  5. Important: Uncheck "Privilege Separation" if you want full admin access, or ensure the token has PVEVMAdmin, PVEAuditor, and Sys.Audit permissions.
  6. Click Add and copy the Secret value immediately. You won't see it again!

Step 2: Add Proxmox Node to WolfStack

Once you have your API Token, add the server to WolfStack:

  1. Open your WolfStack Dashboard.
  2. Click + Add Server in the sidebar.
  3. Select Node Type: üü† Proxmox VE.
  4. Enter the IP Address and Port (default 8006) of your Proxmox node.
  5. Fill in the Token ID (e.g., root@pam!wolfstack) and the Token Secret you copied earlier.
  6. (Optional) Enter a Cluster Name to group nodes in the sidebar.
  7. Click Add Server.
Note: WolfStack will automatically discover other nodes in the same Proxmox cluster if they are reachable via the same API token.
