API Reference

HTTP API and CLI commands

HTTP API

Default: http://localhost:8080

Method Endpoint Description
POST /write/insert Insert row
POST /write/update Update row
POST /write/delete Delete row
POST /write/ddl Execute DDL
GET /health Health check
GET /status Node status
GET /cluster Cluster info
GET /stats Throughput metrics

wolfctl CLI

Command Description
list servers Show cluster nodes
status Local node status
stats Live throughput
promote Request leader promotion
demote Step down
binlog-setup Auto-detect binlog

MySQL Proxy

Connect on port 3307:

mysql -h wolfscale-host -P 3307 -u user -p
