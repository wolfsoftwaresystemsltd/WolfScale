Overview

The Cron Jobs page lets you view, add, edit, and delete cron jobs on any node ‚Äî all from the web UI. WolfStack reads and writes the system crontab directly, so everything you configure here persists across reboots.

üìã Viewing Cron Jobs

All existing cron entries are displayed in a table showing the schedule, command, and optional comment. Each entry can be individually enabled/disabled, edited, or deleted. A collapsible Raw Crontab section shows the complete raw file for advanced users.

‚ûï Adding a Cron Job

Click Add Cron Job and either choose a preset schedule (hourly, daily, weekly, monthly) or enter custom cron syntax (minute hour day month weekday). Then specify the command and an optional comment.

Cron Schedule Format
# minute hour day month weekday
0 3 * * *     # Daily at 3:00 AM
0 */6 * * *   # Every 6 hours
*/5 * * * *   # Every 5 minutes
0 2 * * 1     # Weekly Monday 2:00 AM
